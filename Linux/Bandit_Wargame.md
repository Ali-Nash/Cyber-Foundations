### Commands Learned & Used
* ls: List information about files (current directory by default).
* cd: Change shell working directory.
* cat: Read file data.
* file: Tells you what a file really is.
* du: Tells how much disk space something is using.
* find: Used to find a file or directory based on name, size, etc.
